Exif Tools
====

Command line tools for manipulate Exif(Exchangeable image file format) information. 

# Setup

## Requirements

* Python 3.x

## Install require modules

```
> pip install pillow
> pip install opencv-python
> pip install prettytable
```

# Usage

## Show Exif

```
> ./show-exif [DIR|FILE]
```
